# Week 2 - Internship Tasks

## Overview:

This week focuses on setting up a Python virtual environment, installing required ML libraries, and performing object detection using YOLO.

---

## Task 1: Virtual Environment Setup (venv)

### Objective:

Create and activate a Python virtual environment.

### Steps:

- Created virtual environment using venv

- Activated the environment successfully

### Commands Used:

python -m venv myenv

myenv\Scripts\activate

### Output:

Virtual environment created and activated successfully.

---

## Task 2: Installing Required Package (Ultralytics)

### Objective:

Install ultralytics package for YOLO object detection.

### Command Used:

pip install -U ultralytics

### Output:

Package installed successfully inside virtual environment.

---

## Task 3: YOLO Object Detection

### Objective:

Perform object detection using pretrained YOLO model.

### Output:

Uploaded detected images showing objects identified by YOLO model.

---

## Conclusion:

Successfully completed environment setup, dependency installation, and object detection using YOLO.
