# Week 3 - Task 1: Semantic Segmentation

## Objective:

To perform semantic segmentation using a pretrained YOLO segmentation model.

---

## Concept:

Semantic segmentation is a computer vision technique where each pixel in an image is classified into a specific category. Unlike object detection, which provides bounding boxes, segmentation gives precise object boundaries.

This helps in applications such as:

- Medical image analysis (tumor boundary detection)

- Autonomous vehicles (road and object understanding)

- Robotics and surveillance systems

---

## Process:

- Used a pretrained YOLO segmentation model

- Applied segmentation on input video/images

- Generated segmented output video using model predictions
